Running Ralph Game Package
=========================

Instructions:
1. Install Python 3.x and Pygame on your Windows PC if not installed.
2. Open command prompt in this folder.
3. Run 'python main.py' to start the game.
4. To create a standalone executable (optional), use PyInstaller:
   pip install pyinstaller
   pyinstaller --onefile --windowed main.py
5. Copy the generated exe from dist/ folder along with the assets folder and run_game.bat to your USB stick.
6. Run 'run_game.bat' on any compatible Windows PC to play Running Ralph.

Controls:
- Arrow keys to move Ralph
- Spacebar to jump
- P to pause/resume
- Enter initials on Game Over to save your score

Enjoy!
